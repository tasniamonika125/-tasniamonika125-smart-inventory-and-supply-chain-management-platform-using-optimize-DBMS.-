# Smart-inventory-and-supply-chain-management-platform-using-optimize-DBMS

## Overview    

This project is a web system that helps manage products, stock, and suppliers.It stores all data in a database, so information is organized and easy to update.It helps businesses track inventory quickly and safely. 

## Features

- Add new products with quantity and supplier details.
- Update product information.
- Delete products from inventory.
- View all inventory items in a table format.

## Tech Stack
HTML – structure
CSS – design
JavaScript – buttons, actions

## Database
MySQL 
Tables: products, suppliers, users
